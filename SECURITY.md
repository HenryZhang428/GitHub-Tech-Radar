## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do NOT create a public issue.** Publicly reporting a vulnerability can put the community at risk.
2.  Send an email to [henryzhang428@example.com](mailto:henryzhang428@example.com) (Please replace with your actual email).
3.  Include a detailed description of the vulnerability, steps to reproduce, and potential impact.

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
